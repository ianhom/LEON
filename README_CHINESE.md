## 简介
***LEON*** (Lite Engine Of Network--轻量级网络引擎)聚焦于各类协议（*TCP/IP, ZigBee, 6LowPAN, LoRaWAN...*）的通讯协议栈实现。       
***LEON*** 同时也是我儿子的名字:smile:

## 特点
- 轻量级，适用于资源首先的MCU平台
- 支持无操作系统或RTOS
